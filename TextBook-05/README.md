## 주요 용어 정의 (법 제2조)

| 용어 | 정의 |
|------|------|
| **인공지능** | 데이터 또는 정보를 활용하여 학습·추론·판단 등을 수행하는 기술 또는 시스템 |
| **인공지능시스템** | 인공지능기술을 적용하여 개발된 시스템 |
| **고영향 인공지능** | 국민의 생명·신체·재산 등에 중대한 영향을 미칠 수 있는 AI 시스템 |
| **생성형 인공지능** | 텍스트, 이미지, 영상, 음성 등의 콘텐츠를 생성하는 AI 시스템 |
| **AI사업자** | AI 시스템을 개발·제공·이용하는 자 |
| **인공지능윤리** | 인간 존엄성 기반, 안전·신뢰 확보를 위한 윤리적 기준 |
